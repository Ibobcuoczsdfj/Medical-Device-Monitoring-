 # Smart Platform for Monitoring Medical Equipment Health

## Project Overview
Developing a smart platform that monitors the performance of medical devices by analyzing operational data such as temperature, operating time, and usage patterns.

## Core Innovation
- **AI Integration:** Predicting maintenance needs before failures occur using Predictive Maintenance concepts.
- **Health Indicator:** Generating a numerical health score for each device to prioritize maintenance.

## Technologies
- **Language:** Python
- **AI Models:** Scikit-learn (Random Forest / Regression)
- **Interface:** Web-based Dashboard
